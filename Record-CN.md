#### 1.`pod install`时报错
[官方论坛讨论地址](https://developer.apple.com/forums/thread/652109)
```markdown
解决办法：当需要使用`pod install`的时候，通过Rosetta模式启动命令行来执行pod安装（按住 Command + I 键然后点击命令行图标，这样子应该就可以正常运行了）
```

#### 2.如何引导进入Recovery分区（也许只是DTK需要这样）
```markdown
1. 完全关闭DTK
2. 按住开机按钮直到系统指示灯由白色变成橘黄色
```
